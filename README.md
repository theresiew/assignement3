# Campus Information Website

## Project Description
This project is a single-page campus information website designed to demonstrate HTML structure and CSS layout techniques.

## Layout Behaviors Demonstrated
- Flexbox for navigation and service cards
- CSS Grid for image gallery layout
- Sticky navigation bar
- Fixed announcements notice
- Hidden elements using display property
